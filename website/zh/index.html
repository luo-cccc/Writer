▸核心功能

长上下文

围绕 DeepSeek V4 上下文能力构建，并为长篇写作项目提供压缩路径。

章节流程

初始化项目计划、生成章节草稿，并追踪多章节运行的验收门槛。

记忆感知检索

用摘要、候选清理和项目记忆降低陈旧上下文与检索噪声。

完整工具集

文件操作、Shell、Git、搜索、补丁应用、子智能体和 MCP 服务器仍可使用。

可控模式

Plan、Agent 和自动批准流程适配需要审查的写作与实现任务。

持久化会话

在长时间写作运行中持久化会话、配置、技能、记忆和排队任务。

◎中国大陆镜像安装指南

如果从 GitHub 或 npm 下载较慢，请按以下方式选择最适合你的安装路径：

源码检出

首个 Writer release 发布预编译资产前，优先从源码构建：

git clone https://github.com/luo-cccc/Writer.git
cd Writer
cargo build --release
./target/release/deepseek --version
Cargo + 清华 TUNA 镜像

在 ~/.cargo/config.toml 中添加镜像配置：

[source.crates-io]
replace-with = "tuna"

[source.tuna]
registry = "sparse+https://mirrors.tuna.tsinghua.edu.cn/crates.io-index/"

然后从检出的仓库构建或安装两个二进制文件（调度器在运行时会自动调用 TUI）：

cargo install --path crates/cli --locked
cargo install --path crates/tui --locked
deepseek --version
从源码构建（Rustup 镜像）

如果还没有安装 Rust，先通过清华镜像安装 rustup：

export RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup
export RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

配置 Cargo 镜像后从源码构建：

git clone https://github.com/luo-cccc/Writer.git
cd Writer
cargo install --path crates/cli --locked
cargo install --path crates/tui --locked
手动下载预编译二进制

直接从 GitHub Releases 下载对应平台的二进制文件，放到 PATH 目录即可：

mkdir -p ~/.local/bin
curl -L -o ~/.local/bin/deepseek \
  https://github.com/luo-cccc/Writer/releases/latest/download/deepseek-linux-x64
curl -L -o ~/.local/bin/deepseek-tui \
  https://github.com/luo-cccc/Writer/releases/latest/download/deepseek-tui-linux-x64
chmod +x ~/.local/bin/deepseek ~/.local/bin/deepseek-tui

macOS 用户将 linux-x64 替换为 macos-arm64 或 macos-x64，并将 sha256sum 替换为 shasum -a 256。

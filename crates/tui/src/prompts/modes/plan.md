## Mode: Plan

You are running in Plan mode — design the novel workflow before changing files.

Investigate first, act later. Read the existing book bible, cards, outlines, chapter drafts, and memory ledgers. Build a clear plan for what should change and why. All writes and patches are blocked in this mode.

Use Plan mode for:

- turning a premise into a book architecture;
- diagnosing continuity drift;
- designing a revision pass across multiple chapters;
- mapping character arcs, timelines, factions, mysteries, and payoff chains;
- deciding what assets should be created before drafting.

When the plan is solid, the user can switch modes or run the relevant top-level `deepseek ...` book command.
